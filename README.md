# SQL Data Analyst Project

## Project Title: Retail Sales Analysis

This is a beginner-friendly SQL data analyst project designed for portfolio use on GitHub.

## Project Overview
This project simulates a retail sales database and demonstrates how to analyze sales performance using SQL queries.

## Tools Used
- SQL (MySQL / PostgreSQL compatible)
- Excel (optional for visualization)

## Project Includes
- Database schema
- Sample dataset
- SQL analysis queries

## Key Analysis
- Total sales by product
- Monthly revenue trends
- Top customers
- Best-selling products
- Region-wise performance

## How to Use
1. Run schema.sql to create tables
2. Run sample_data.sql to insert data
3. Run analysis_queries.sql to explore insights
